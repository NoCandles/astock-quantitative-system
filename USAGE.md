# A股量化系统 - 使用指南

## 📋 系统简介

这是一个基于 Web 的 A股量化交易辅助系统，可以帮助你：

- 🔍 搜索和添加股票到自选列表
- 📊 查看股票实时行情
- 📈 查看多种周期的K线图表
- 🤖 获取 AI 智能诊断分析

## 🚀 快速开始

### 方法一：使用启动脚本（推荐 Windows 用户）

1. 双击运行 `start.bat` 文件
2. 系统会自动安装依赖并启动服务
3. 等待浏览器自动打开 http://localhost:3000

### 方法二：手动启动

#### 1. 启动后端服务

```bash
# 进入后端目录
cd backend

# 安装依赖
npm install

# 启动服务
npm start
```

后端服务将运行在 http://localhost:5000

#### 2. 启动前端服务（另开一个终端窗口）

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端服务将运行在 http://localhost:3000

#### 3. 访问系统

打开浏览器访问 http://localhost:3000

## 📖 功能使用

### 1. 添加自选股票

1. 在首页点击"自选股"菜单
2. 在搜索框中输入股票代码或名称（如：600519 或 贵州茅台）
3. 从下拉列表中选择股票
4. 点击"添加"按钮
5. 股票将显示在自选列表中

### 2. 查看股票详情

1. 在自选列表中点击任意股票卡片
2. 进入股票详情页面
3. 查看实时行情和K线图表

### 3. K线图表操作

#### 切换时间周期
- **分时**：显示当日分时走势
- **5日**：最近5个交易日K线
- **3日**：最近3个交易日K线
- **日线**：每日K线（默认）
- **周线**：每周K线
- **月线**：每月K线
- **年线**：每年K线

#### 图表交互
- **缩放**：鼠标滚轮放大/缩小
- **平移**：点击拖动图表
- **查看数据**：鼠标悬停查看具体数值
- **均线显示**：自动显示 MA5、MA10、MA20 均线

### 4. AI 智能诊断

1. 在股票详情页面
2. 点击"开始分析"按钮
3. 等待 AI 分析完成
4. 查看分析结果，包括：
   - 短期趋势判断
   - 技术面分析
   - 风险提示
   - 操作建议

## ⚙️ 配置说明

### AI 诊断配置（可选）

系统内置了模拟分析功能，也可以配置真实的 AI 分析：

1. 获取 Claude API Key
2. 在 `backend` 目录下创建 `.env` 文件
3. 添加以下内容：
   ```
   CLAUDE_API_KEY=your_api_key_here
   PORT=5000
   ```

### 数据刷新

- 自选股行情：每 5 秒自动刷新
- K线数据：每次切换周期时更新
- AI 分析：手动触发

## 📊 数据说明

### 股票代码
- 上交所：6开头（如：600519 贵州茅台）
- 深交所：0开头（如：000858 五粮液）

### 行情数据
- **价格单位**：元
- **成交量单位**：股/手
- **涨跌幅**：百分比显示
- **颜色标识**：
  - 🔴 红色：上涨
  - 🟢 绿色：下跌
  - ⚪ 灰色：平盘

### K线数据
- **开盘价**：当日开盘价格
- **收盘价**：当日收盘价格
- **最高价**：当日最高价格
- **最低价**：当日最低价格
- **成交量**：当日成交股数

## 🔧 技术指标说明

### 均线系统 (MA)
- **MA5**：5日移动平均线，短期趋势
- **MA10**：10日移动平均线
- **MA20**：20日移动平均线，中期趋势
- **MA60**：60日移动平均线，长期趋势

### RSI 指标
- **超买区**：RSI > 70，可能回调风险
- **超卖区**：RSI < 30，可能反弹机会
- **正常区**：RSI 30-70

## 🐛 常见问题

### Q: 启动失败怎么办？
A: 
- 检查 Node.js 版本（需要 18+）
- 确保端口 3000 和 5000 未被占用
- 检查依赖是否安装成功

### Q: 股票搜索不到？
A:
- 确认股票代码是否正确
- 尝试只输入部分代码或名称
- 检查网络连接

### Q: K线图加载失败？
A:
- 刷新页面重试
- 检查网络连接
- 可能是数据源临时不可用

### Q: AI 分析失败？
A:
- 系统会自动降级到模拟分析
- 如需真实 AI 分析，请配置 API Key
